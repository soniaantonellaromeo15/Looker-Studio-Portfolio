# Looker Studio Data Analytics Portfolio 📊
## Raccolta di Soluzioni di Business Intelligence e Data Visualization

Benvenuto nella mia repository dedicata ai progetti di Data Visualization sviluppati su **Looker Studio**. Questa cartella raccoglie dashboard interattive progettate per trasformare dataset grezzi in insight strategici e visivi, applicando logiche di data storytelling e un design focalizzato sulla UI/UX.

---

## 🎬 1. Academy Awards (Premi Oscar) Dashboard
Analisi approfondita dello storico dei premi cinematografici più prestigiosi al mondo. La dashboard esplora le evoluzioni delle premiazioni, i record individuali e le tendenze delle case di produzione nel corso degli anni.

* **Strumenti:** Looker Studio, Google Sheets.
* **Metriche Chiave:** Distribuzione dei premi per genere, attori/registi più premiati, evoluzione temporale delle nomination.
* 🔗 **[Naviga la Dashboard Interattiva degli Academy Awards](https://datastudio.google.com/u/0/reporting/38f49043-603d-4de1-a85e-c9f9398bf3a5/page/p_9h5oysjq1d)**

## 🎤 2. Festival di Sanremo (1951–2023) Analysis
Un viaggio analitico attraverso oltre 70 anni di storia della musica italiana. Il report mappa l'evoluzione dei partecipanti, dei vincitori e dei trend musicali che hanno caratterizzato il Festival dal 1951 al 2023.

* **Strumenti:** Looker Studio, Excel / CSV Dataset.
* **Metriche Chiave:** Trend storici delle canzoni in gara, artisti con maggior numero di podi e presenze, analisi delle categorie.
* 🔗 **[Naviga la Dashboard Interattiva di Sanremo](https://datastudio.google.com/u/0/reporting/1f516bc1-87b1-461c-8508-e83de74cb057/page/O7crF)**

---

### 📊 Caratteristiche Principali delle Dashboard
* **Filtri Dinamici:** Permettono all'utente di esplorare i dati in tempo reale per anno, artista o categoria.
* **Data Storytelling:** Grafici ottimizzati per mostrare l'evoluzione dei macro-trend nel corso del tempo.
* **Interfaccia Pulita:** Layout strutturato per garantire una navigazione fluida e un impatto visivo immediato.
